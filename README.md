# Prospira – Smarter Job Search with AI

An AI-powered job search platform that matches your resume with the best job opportunities tailored to your skills and goals.

## Features

- **Smart Resume Matching**: Upload your resume and let our AI match you with relevant job opportunities
- **AI Tools Chat**: Get AI assistance with resume tips and job search strategies
- **Bias-Free Matching**: Our algorithm ensures fair and accurate job matches without bias
- **Get Started**: Simple onboarding process to start your job search

## Deployment

This site is deployed using GitHub Pages. Any push to the main branch will automatically update the live site.

### To deploy:
1. Push your changes to GitHub
2. Go to **Settings** → **Pages**
3. Select **Deploy from a branch** with the `main` branch and `/root` folder
4. Your site will be live in a few minutes
